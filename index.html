What is Ionic?

Ionic Framework is an open source UI toolkit that focused on the user experience of an app (such as the controls, interactions, gestures, and animations). The framework provides pre-designed components which allows for easily applicable themes (given a platform).


Ionic makes it easy to build high-performance mobile and Progressive Web Apps (or PWAs) that work on any platform or device.

PWA's= web apps that load like regular web pages but can offer functionality such as working offline, push notifications, device hardware access, and app-style navigation
So who uses this?

Ionic is one of the most popular open source app platforms in the world! It was launched in 2013 and 5M+ apps were created by 5M+ developers in over 200 countries.


The creators of the framework hoped to streamline the process of building mobile apps that align to modern Web standards. Many groups rely on ionic to build sleek apps. Such groups include Amtrak, Nationwide, and NASA.

Angular is a popular JS framework which provides the developer with the right tools to build a web app while also defining how the code should be organized. Many modern websites rely heavily on JS.


Ionic provides a collection of components that mimic the native look, feel and functionality of each mobile platform. Examples of these components include buttons, tabs, menus, lists, cards, modals, and so on.


HTML, CSS, and JS are then used to add whatever styles and features you want to your site.


Cordova is yet another framework that makes it easy to create an app that runs a web app in a webview. Ionic uses it to build and run apps as native apps on each device.

So how do we set this up?
Setting up is easy, it should only take a few minutes!

You will need a Node and npm environment. Node is a runtime environment that allows JS to be written on the server-side. It is often used to build dev tools such as the Ionic CLI (Command Line Interface). Npm is the package manager for node; it allows developers to install, share, and package node modules. Ionic can be installed with npm.
